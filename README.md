# Journalism

## Installation

### Start

```bash
docker-compose up
```

### Stop

```bash
docker-compose down
```

### Build

```bash
docker-compose build
```

Note: "docker-compose up" automatically builds/rebuilds the project

## Usage

Open your web browser @ <http://localhost:80/>
